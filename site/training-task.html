Training task

We will now use the example of the laptop stand to get familiar with both tools that will be used in this study. We will give a task to build a parametric specification similar to the actual tasks that will be performed in this study. We will show how this training task is performed in both tools. You can then replicate these procedures yourself to get familiar with both tools. Once you feel confident with a tool to start the task you're assigned to, you can move to the actual task.

Task

For this task we will focus on a partial model of the laptop stand consisting of a side leg and the top plate. We will start with a partially constrained design consisting of 2 sketches and 2 extrusions as illustrated below.

The training task consists of correctly specifying parametric behavior to ensure this part resizes correctly when resized to fit laptops of different dimensions. The desired parametric behavior is defined as follows:

  1. When the depth of the laptop that needs to be accommodated by the laptop stand changes, the material of the side leg should remain of uniform thickness. You can test this by changing the intended depth of the laptop as shown below. (not sure if clear that we mean to make top parallel and left-bottom perp)
  2. When the depth of the laptop changes, the width of the tabs should resize accordingly. The space between the tabs should always be twice the width of a tab. You can test this by changing the intended depth as in task (1).
  3. When the material thickness changes with which the laptop stand is intended to be fabricated, the size of all tabs should update to accommodate the new material thickness. More specifically, the size of all tabs should follow that of the top-left tab. You can test this by changing the size of the top-left tab as shown below.
  4. As an extension to task (3), not only the size of the tabs should change, but the thickness of the top plate (in 3D) should also scale correctly with the thickness to ensure a proper fitment in the tabs.
